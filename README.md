# Antigravity Apex

An elite, AI-native desktop environment built on Next.js. Engineered for high-velocity operators who demand performance, precision, and cinematic execution.

## Features

- **OS-Style Interface**: A fully functional window management system with a dock, menu bar, and desktop icons.
- **AI-Native Core**: Integrated intelligence across strategy, creative, and engineering workflows.
- **Cinematic Motion**: Smooth transitions and editorial-grade design using Framer Motion.
- **Next.js & Tailwind**: Built on the latest web technologies for millisecond-level responsiveness.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm or npm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Radix UI / Shadcn
